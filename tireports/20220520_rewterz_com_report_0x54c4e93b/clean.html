Analysis Summary

CVE-2022-21151

Intel Processors could allow a local authenticated attacker to obtain sensitive information, caused by the processor optimization removal or modification of security critical code. By sending a specially-crafted request, an attacker could exploit this vulnerability to obtain sensitive information.
