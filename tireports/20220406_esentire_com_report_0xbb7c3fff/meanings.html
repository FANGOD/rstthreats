This gives the threat actor(s) a huge benefit as they can change or not have to rely on IP addresses to avoid detection.
  • The malware hashes the libraries and API functions by applying ROR-13 calculation to evade detections. GetProcAddress API is used to resolve the functions address. (Exhibit 8-9).
