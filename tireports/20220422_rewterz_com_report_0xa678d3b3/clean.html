Analysis Summary

CVE-2022-24858

Node.js next-auth module could allow a remote attacker to conduct phishing attacks, caused by an open redirect vulnerability. An attacker could exploit this vulnerability using a specially-crafted UR to redirect a victim to arbitrary Web sites.

Impact

Remote Code Execution
