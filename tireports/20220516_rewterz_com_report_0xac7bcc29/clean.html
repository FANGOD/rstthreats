Analysis Summary

CVE-2022-29108

Microsoft SharePoint Server could allow a remote authenticated attacker to execute arbitrary code on the system, caused by a flaw in the SharePoint Server component. By sending a specially-crafted request, an attacker could exploit this vulnerability to execute arbitrary code on the system.

Impact

  • Code Execution
