Analysis Summary

CVE-2022-28193

NVIDIA Jetson Linux Driver Package could allow a local attacker to execute arbitrary code on the system, caused by a memory buffer overflow in Cboot module tegrabl_cbo.c. By executing a specially-crafted program, an attacker could exploit this vulnerability to execute arbitrary code on the system.
