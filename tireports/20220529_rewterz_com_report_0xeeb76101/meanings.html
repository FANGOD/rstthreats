Sentences (1)
  • Malspam is being used to target victims in an Agent Tesla campaign. Since its initial appearance in 2014, this has been deployed in many forms, most notably via phishing attempts. AgentTesla is renowned for stealing data from a variety of target workstations apps, including browsers, FTP clients, and file downloaders. Agent Tesla grabs data from the victims clipboard, logs keystrokes, captures screenshots, and gains access to the victims webcam. It has the ability to terminate running analytic programs and anti-virus applications. In an attempt to disguise its capabilities and activities from researchers, the malware also runs simple checks to see if it is operating on a virtual machine or in debug mode.
