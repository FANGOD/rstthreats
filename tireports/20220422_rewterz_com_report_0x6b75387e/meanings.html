References:

Sentences (1)
  • Racoon gathers private data such as credit card numbers, cryptocurrency wallet addresses, login passwords, and browser information like cookies and history. It was discovered in the wild for the first time in April 2019. This malware is a C++ program that allows you to steal data from common browsers and cryptocurrency wallets. Exploit kits are used to deploy the virus, which employs browser-based vulnerabilities to lead users to landing sites containing exploit codes. Its also propagated through phishing efforts that persuade victims to run the malicious payload or macros. This info stealer collects related to the machine, such as the OS arch and version, hardware information, and installed apps.
