Analysis Summary

CVE-2021-36828

WP Maintenance plugin for WordPress is vulnerable to cross-site scripting, caused by improper validation of user-supplied input. A remote authenticated attacker could exploit this vulnerability using to inject malicious script into a Web page which would be executed in a victim’s Web browser within the security context of the hosting Web site, once the page is viewed. An attacker could use this vulnerability to steal the victim’s cookie-based authentication credentials.

Impact

Cross-Site Scripting

Affected Products

  • WordPress WP Maintenance plugin for WordPress 6.0.4
  • WordPress WP Maintenance plugin for WordPress 6.0.3

Remediation

Upgrade to the latest version of WP Maintenance plugin for WordPress, available from the WordPress Plugin Directory.
