Analysis Summary

CVE-2022-22966

VMware Cloud Director could allow a remote authenticated attacker to execute arbitrary code on the system, caused by a flaw in the VMware Cloud Director tenant. By sending a specially-crafted request, an attacker could exploit this vulnerability to execute arbitrary code on the system.
